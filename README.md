# 트립파인더 (TripFinder)

Trip.com URL의 국가 서브도메인만 변경하여 여러 국가별 링크를 생성하는 웹 도구입니다.

## 기능

- ✅ Trip.com URL 입력 시 모든 국가별 링크 자동 생성
- ✅ 최근 검색 기록 저장 (localStorage)
- ✅ 잘못된 URL 검증 및 오류 메시지 표시
- ✅ 사용방법 안내 모달
- ✅ 반응형 디자인

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 사용 방법

1. Trip.com URL을 입력창에 붙여넣습니다 (예: `https://kr.trip.com/flights/showfarefirst?locale=ko-KR&curr=KRW`)
2. "생성" 버튼을 클릭하거나 Enter 키를 누릅니다
3. 모든 국가별 링크가 자동으로 생성됩니다
4. 링크를 클릭하여 새 탭에서 열거나 "복사" 버튼으로 URL을 복사할 수 있습니다

## 지원 국가

대한민국, 베트남, 일본, 태국, 미국, 스페인, 프랑스, 독일, 캐나다, 호주, 네덜란드, 싱가폴, 인도네시아, 말레이시아, 대만, 인도, 멕시코, 영국, 러시아, 포르투갈, 사우디

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3

## 주의사항

- 이 도구는 단순 URL 생성 도구이며, 크롤링이나 가격 계산을 하지 않습니다
- 국가 서브도메인만 변경되며, 경로와 모든 파라미터는 그대로 유지됩니다
- 최근 검색 기록은 브라우저의 localStorage에 저장됩니다
